monoampmidi

monoampmidi — Genera un'ampiezza per sintetizzatori monofonici basata sulla velocity MIDI.

Description

monoampmidi converte la velocity MIDI in un valore di ampiezza adatto per sintetizzatori monofonici. Funziona in combinazione con il sistema di gestione MIDI mono, permettendo di controllare l'ampiezza iniziale e il comportamento di retrigger quando vengono suonate note legato.

Syntax

iamp monoampmidi ivelamp, iretrigger

Initialization

ivelamp -- Fattore di moltiplicazione della velocity (0-1). Controlla quanto la velocity MIDI influenza l'ampiezza finale.

iretrigger -- Modalità retrigger: 0 = no retrigger, 1 = retrigger su note legato. Quando attivo, ogni nuova nota riavvia l'ampiezza anche se ci sono già note premute.

Performance

L'ampiezza viene calcolata in base allo stato delle note MIDI:

  • Prima nota: amp = (velocity / 127) * ivelamp
  • Note legato senza retrigger: l'ampiezza non cambia
  • Note legato con retrigger: l'ampiezza viene ricalcolata e riavviata.

Nota
Quando si usa monoampmidi, l'envelope ADSR (se presente) parte sempre da 0. L'ampiezza di monoampmidi rappresenta il livello di "sustain" raggiunto dall'envelope.

Examples

Ecco un esempio dell'opcode monoampmidi.

Example 1. Uso base

instr 1
    ; Ampiezza semplice dalla velocity
    iamp monoampmidi 0.8, 0
    kfreq monocpsmidi 0.1, 0
    
    asig vco2 iamp, kfreq
    outs asig, asig
endin
