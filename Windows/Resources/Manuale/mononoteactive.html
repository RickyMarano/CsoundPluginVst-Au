mononoteactive

mononoteactive — Restituisce 1 se c'è almeno una nota attiva, 0 altrimenti.

Description

mononoteactive fornisce un segnale binario (gate) che indica se ci sono note MIDI attualmente premute nel sistema monofonico. È uno strumento essenziale per trigger condizionali, reset di oscillatori/LFO o per attivare effetti solo durante l'esecuzione.

Syntax

kactive mononoteactive

Performance

L'opcode monitora costantemente lo stack delle note. Restituisce 1.0 finché almeno un tasto è premuto; restituisce 0.0 non appena l'ultima nota viene rilasciata.

Nota
Questo opcode differisce da monoAdsr: mentre un inviluppo può rimanere attivo durante la coda di release, mononoteactive scende a 0 istantaneamente al "note-off" fisico dell'ultima nota.
Aspetto mononoteactive monoAdsr
Risposta al note on Passa immediatamente a 1 Sale gradualmente (Attack)
Risposta al note off Passa immediatamente a 0 Scende gradualmente (Release)

Examples

Ecco un esempio di utilizzo come gate per un generatore di rumore.

Example 1. Noise Gate condizionale

instr 1
    kfreq   monocpsmidi 0.1, 0
    iamp    monoampmidi 0.8, 0
    kactive mononoteactive
    
    ; Il rumore si sente solo quando il tasto è fisicamente premuto
    anoise  noise iamp * 0.1, 0
    asig    vco2 iamp, kfreq
    
    asig    = asig + (anoise * kactive)
    outs    asig, asig
endin
