monoaftertouch

monoaftertouch — Legge il channel aftertouch (channel pressure) MIDI normalizzato 0-1.

Description

monoaftertouch converte i messaggi MIDI di channel aftertouch (pressione del canale) in un valore normalizzato da 0 a 1. L'aftertouch è la pressione applicata ai tasti dopo il contatto iniziale, utilizzata per aggiungere espressività dinamica alla performance in tempo reale.

Syntax

kaft monoaftertouch

Performance

L'opcode restituisce il valore aftertouch / 127. Il valore viene aggiornato istantaneamente ogni volta che il controller MIDI invia un nuovo messaggio di pressione del canale.

Nota sulla compatibilità
Questo opcode gestisce esclusivamente il channel aftertouch (pressione globale del canale). Non supporta il polyphonic aftertouch (pressione individuale per tasto), che richiede opcode specifici.

Examples

Esempio di modulazione della risonanza tramite pressione del tasto.

Example 1. Aftertouch-controlled Resonance

instr 1
    kfreq monocpsmidi 0.1, 0
    iamp  monoampmidi 0.8, 0
    aenv  monoAdsr 0.01, 0.1, 0.7, 0.2, 0, 0, 0, 0
    kaft  monoaftertouch
    
    ; La risonanza aumenta premendo il tasto più a fondo
    kres  = 0.2 + (kaft * 0.7)
    
    asig  vco2 iamp, kfreq
    asig  moogladder asig, 2000, kres
    asig  = asig * aenv
    outs  asig, asig
endin
