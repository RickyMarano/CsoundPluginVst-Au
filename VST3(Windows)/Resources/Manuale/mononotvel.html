mononotvel

mononotvel — Restituisce la velocity MIDI normalizzata (0-1) dell'ultima nota suonata.

Description

mononotvel fornisce la velocity dell'ultima nota MIDI suonata, normalizzata in un intervallo da 0 a 1. A differenza di monoampmidi, questo opcode restituisce il valore di velocity puro, senza moltiplicatori i-rate, rendendolo ideale per modulazioni di parametri indipendenti dall'ampiezza (come il cutoff di un filtro o il tempo di attacco).

Syntax

kvel mononotvel

Performance

L'opcode calcola internamente il rapporto velocity / 127 in base all'ultima nota registrata nel sistema monofonico.

Opcode Output Uso tipico
monoampmidi (velocity / 127) * ivelamp Controllo ampiezza principale
mononotvel velocity / 127 Modulazioni (filtro, PWM, inviluppi)

Examples

Ecco un esempio di modulazione del filtro basata sulla velocity.

Example 1. Velocity-to-Filter Cutoff

instr 1
    kfreq monocpsmidi 0.1, 0
    iamp  monoampmidi 0.8, 0
    aenv  monoAdsr 0.01, 0.1, 0.7, 0.2, 0, 0, 0, 0
    
    ; Velocity alta = filtro più aperto
    kvel  mononotvel
    kcutoff = 500 + (kvel * 4000)
    
    asig  vco2 iamp, kfreq
    asig  moogladder asig, kcutoff, 0.6
    asig  = asig * aenv
    outs  asig, asig
endin
