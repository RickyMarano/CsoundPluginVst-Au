Description

monocpsmidi converte note MIDI in frequenze in Hz con supporto per glide (portamento) tra le note e pitch bend MIDI opzionale integrato. Il glide puÃ² essere modellato con diverse curve (lineare, esponenziale, logaritmica).

Syntax

kfreq monocpsmidi iglide, iglidepow [, ipbmode, ipbrange]

Initialization

iglide -- Tempo di glide in secondi (0 = nessun glide, valori tipici: 0.05 - 0.5).

iglidepow -- Forma della curva di glide: 0 = lineare, >0 = esponenziale, <0 = logaritmica. Valori tipici: da -2 a +2.

ipbmode (optional, default=0) -- ModalitÃ  pitch bend: 0 = disabilitato, 1 = offset Hz, 2 = ratio (consigliato).

ipbrange (optional, default=2) -- Range del pitch bend in semitoni.

Performance

Ad ogni k-cycle, l'opcode calcola la frequenza corrente applicando:

  • Glide: transizione fluida dalla frequenza precedente alla nuova nota.
  • Pitch bend: modulazione in tempo reale della frequenza (se abilitato).

Curve di glide
iglidepow = 0: Lineare (velocitÃ  costante)
iglidepow > 0: Esponenziale (parte lenta, finisce veloce)
iglidepow < 0: Logaritmica (parte veloce, finisce lenta)
