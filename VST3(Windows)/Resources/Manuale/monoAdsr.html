monoAdsr

monoAdsr — Genera un envelope ADSR (Attack-Decay-Sustain-Release) per sintetizzatori monofonici con curve personalizzabili.

Description

monoAdsr genera un envelope classico ADSR sincronizzato con il sistema di gestione MIDI mono. L'envelope si attiva quando viene suonata la prima nota e passa in fase di release quando tutte le note sono rilasciate. Ogni fase (attack, decay, release) può avere una curva di risposta personalizzata.

    │    ╱  ╲___________
    │   ╱    ▀▀▀▀▀      ╲
sus │  ╱              ▀▀▀╲
    │ ╱                    ╲___
0.0 ┼─────────────────────────────
    └─A──D───S────────────R───→
        A = Attack, D = Decay, S = Sustain, R = Release

Syntax

aenv monoAdsr iatk, idec, isus, irel, iatkpow, idecpow, irelpow, iretrigger

Initialization

iatk -- Tempo di attack in secondi (tipicamente da 0 a 10).

idec -- Tempo di decay in secondi.

isus -- Livello di sustain (da 0 a 1).

irel -- Tempo di release in secondi.

iatkpow, idecpow, irelpow -- Forma della curva per le rispettive fasi: 0 = lineare, >0 = esponenziale, <0 = logaritmica.

Performance

L'envelope attraversa le seguenti fasi gestite internamente:

  • ATTACK: Sale da 0 a 1 all'attivazione della nota.
  • DECAY: Scende dal picco (1) al livello di sustain stabilito.
  • SUSTAIN: Mantiene il livello finché il gate MIDI è attivo (almeno una nota premuta).
  • RELEASE: Scende dal livello attuale a 0 quando tutte le note vengono rilasciate.

Tip sulle curve
Valori tipici per le potenze (pow) sono compresi tra - 0 + .Curva lineare con 0, le curve esponenziali (+) creano attacchi più "percussivi" (snappy), mentre quelle logaritmiche (-) risultano più morbide.

Examples

Ecco un esempio di un sintetizzatore monofonico completo con monoAdsr.

Example 1. Synth con retrigger legato

instr 1
    kfreq monocpsmidi 0.1, 0
    iamp  monoampmidi 0.8, 1  ; Retrigger ampiezza attivo
    
    ; Inviluppo con attack 10ms e retrigger attivo
    aenv  monoAdsr 0.01, 0.1, 0.7, 0.2, 0, 0, 0
    
    asig  vco2 iamp, kfreq
    asig  = asig * aenv
    outs  asig, asig
endin
