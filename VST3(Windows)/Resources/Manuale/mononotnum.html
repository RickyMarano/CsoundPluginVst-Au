mononotnum

mononotnum — Restituisce il numero della nota MIDI attualmente suonata in modalità mono.

Description

mononotnum fornisce il numero MIDI (0-127) dell'ultima nota suonata nel sistema monofonico. È particolarmente utile per implementare modulazioni basate sul pitch, key tracking del filtro e visualizzazioni di stato.

Syntax

knotnum mononotnum

Performance

L'opcode restituisce il numero MIDI dell'ultima nota presente nello stack monofonico. Se non ci sono note attive (tasti rilasciati), l'output restituisce 0.

Nota Numero MIDI Frequenza (Hz)
C-108.18
A02127.50
C3 (Middle C)60261.63
A469440.00
C81084186.01
G912712543.85

Examples

Ecco un esempio di key tracking per il filtro utilizzando mononotnum.

Example 1. Filtro Key Tracking

instr 1
    kfreq monocpsmidi 0.1, 0
    iamp  monoampmidi 0.8, 0
    aenv  monoAdsr 0.01, 0.1, 0.7, 0.2, 0, 0, 0, 0
    
    ; Il filtro segue la nota suonata
    knotnum mononotnum
    kcutoff = 200 + (knotnum * 40)
    
    asig  vco2 iamp, kfreq
    asig  moogladder asig, kcutoff, 0.6
    asig  = asig * aenv
    outs  asig, asig
endin
