monopitchbend

monopitchbend — Legge il pitch bend MIDI e lo restituisce in diverse modalità.

Description

monopitchbend converte i messaggi MIDI pitch bend in valori utilizzabili per modulare frequenze, filtri o altri parametri. Supporta quattro modalità di output diverse per adattarsi a diverse esigenze di sintesi monofonica.

Syntax

kpb monopitchbend [imode] [, irange]

Initialization

imode (optional, default=0) -- Modalità di output:

  • 0 = normalizzato (-1.0 a 1.0)
  • 1 = semitoni (-irange a +irange)
  • 2 = ratio (moltiplicatore di frequenza)
  • 3 = Hz offset

irange (optional, default=2) -- Estensione del pitch bend in semitoni (tipicamente 2 o 12).

Performance

L'opcode monitora i dati pitch bend in arrivo sul canale MIDI corrente e aggiorna il valore di kpb ad ogni k-cycle.

Consiglio matematico
Per modulare la frequenza di un oscillatore in modo musicalmente coerente, si consiglia di usare il mode 2 (ratio) tramite moltiplicazione (es. kfreq = kfreq * kpb).

Examples

Esempio di utilizzo standard per il controllo del pitch.

Example 1. Controllo Pitch Standard

instr 1
    kfreq monocpsmidi 0.1, 0
    iamp  monoampmidi 0.8, 0
    
    ; Mode 2 (ratio) è la scelta standard per le frequenze
    kpb   monopitchbend 2, 2
    kfreq = kfreq * kpb
    
    asig  vco2 iamp, kfreq
    outs  asig, asig
endin
