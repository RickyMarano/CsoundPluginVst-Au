monocc

monocc — Legge i messaggi MIDI Control Change (CC) per sintetizzatori monofonici.

Description

monocc converte i messaggi MIDI Control Change in valori utilizzabili per modulare parametri del sintetizzatore. Supporta tutti i 128 controller MIDI standard e può restituire valori sia raw (0-127) che normalizzati (0-1).

Syntax

kcc monocc iccnum [, imode]

Initialization

iccnum -- Numero del controller MIDI (0-127).

imode (optional, default=0) -- Modalità di output: 0 = valore raw (0-127), 1 = valore normalizzato (0.0-1.0).

Valori di default all'avvio
All'inizializzazione, alcuni controller hanno valori predefiniti: CC 7 (Volume) = 100, CC 10 (Pan) = 64, CC 11 (Expression) = 127. Tutti gli altri sono impostati a 0.

Performance

L'opcode monitora i dati CC in arrivo e aggiorna il valore kcc. Sotto sono elencati i CC più comuni:

CC# Nome Uso tipico
1Mod WheelVibrato, modulazioni
7VolumeVolume master del canale
10PanStereo (0=L, 64=C, 127=R)
64Sustain PedalOn/Off (soglia 64)
74BrightnessCutoff del filtro

Examples

Esempio di utilizzo della Mod Wheel per controllare il vibrato.

Example 1. Mod Wheel Vibrato

instr 1
    kfreq monocpsmidi 0.1, 0
    iamp  monoampmidi 0.8, 0
    
    ; CC1 (Mod Wheel) normalizzato
    kmod  monocc 1, 1
    
    ; Vibrato depth controllato dalla mod wheel
    kvib  lfo 1, 5
    kfreq = kfreq * cent(kvib * kmod * 20)
    
    asig  vco2 iamp, kfreq
    outs  asig, asig
endin
